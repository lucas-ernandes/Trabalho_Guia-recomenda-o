GUIA DE RECOMENDAÇÃO

Este site foi desenvolvido com o objetivo de apresentar um Guia de Recomendações contendo filmes, séries, animes, mangás e jogos. A página organiza as informações de maneira visual para facilitar a navegação do usuário, permitindo acesso rápido a cada categoria e oferecendo sinopses detalhadas de cada obra recomendada.

A estrutura do site foi construída utilizando HTML, enquanto a apresentação visual, responsividade e design foram realizados com CSS. O HTML foi essencial para aplicar, fontes, tabelas, titulos, listas, links e imagens. CSS foi importante para melhorar visuamente o layout e formatação geral, como, aplicar cores no corpo do site, personalização da tabela, tamanho da imagem, formatar links e aplicar fontes externas.
